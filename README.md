# Description
Use [gnu aspell](http://aspell.net/) on a block of text from line N to M. 

# Use and requirements
- Use: 
```sh
    Usage: 
        [-n from] [-m to] [-f FILENAME]
```

- You need `aspell`
- Just clone the repo
